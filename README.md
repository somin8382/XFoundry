# XFoundry Landing Page

**주식회사 엑스파운더리** 공식 랜딩 페이지입니다.

## About

XFoundry는 데이터와 구조로 성공을 설계하는 사업 생성 인프라입니다.
선수금 0원, 완전 성공보수제로 운영되며, 상위 3%의 창업가만을 선발하여 아이디어부터 IPO/EXIT까지 전 과정을 함께합니다.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (Noto Sans KR)

## Features

- 프리미엄 다크 모드 UI (Gold/Amber 액센트)
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 8단계 인터랙티브 타임라인
- 스크롤 기반 네비게이션 효과
- 글래스모피즘 UI 요소
- 한국어 타이포그래피 최적화 (Negative letter-spacing, word-break: keep-all)

## Deployment

GitHub Pages로 배포할 수 있습니다.

1. 이 레포지토리를 생성합니다.
2. **Settings > Pages**에서 Source를 `main` 브랜치로 설정합니다.
3. `https://<username>.github.io/<repo-name>/` 에서 확인할 수 있습니다.
